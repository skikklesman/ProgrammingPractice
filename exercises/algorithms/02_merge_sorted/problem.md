# Merge Two Sorted Arrays

Given two sorted integer vectors, merge them into a single sorted vector.

## Example

```
Input:  a = {1, 3, 5}, b = {2, 4, 6}
Output: {1, 2, 3, 4, 5, 6}
```

## Goal

Implement `merge_sorted()` in **solution.cpp** so that all tests pass.
Take advantage of the fact that both inputs are already sorted — you should
be able to do this in O(n + m) time with a single pass.
