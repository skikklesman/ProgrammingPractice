# Bug Hunt: Binary Search

`binary_search()` takes a sorted vector and a target value. It should return
the index of the target, or `-1` if the target is not found.

The implementation looks reasonable at a glance, but it has **subtle bugs**
that cause it to return wrong results or behave incorrectly in edge cases.

## Goal

Run the tests, observe the failures, then **fix the bugs** in `solution.cpp`
so that all tests pass.
